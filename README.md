# breaking the block

簡單的打磚塊遊戲，使用 Python 和 Pygame 開發。

## 安裝

建議建立虛擬環境後安裝相依套件：

1. 建立虛擬環境並啟用（macOS / Linux）：

   python3 -m venv venv
   source venv/bin/activate

2. 安裝套件：

   pip install -r requirements.txt

## 執行

執行主程式：

    python main.py

## 開發

- 程式碼位於 `main.py`，未來會拆分到 `src/` 目錄。
- 資源圖片放在 `assets/images/`（或舊有的 `image/`）
